# Projects

建议用下面的命名方式创建各阶段项目目录：

- `project-01-structured-extractor`
- `project-02-tool-faq-agent`
- `project-03-research-agent`
- `project-04-private-rag`
- `project-05-mcp-guardrails`
- `project-06-advanced-experiments`

每个项目目录至少包含：

- `README.md`
- `src/`
- `tests/`
- `artifacts/`

每个项目 README 至少写清楚：

- 问题定义
- 输入输出
- 架构
- 运行方式
- 评测方式
- 已知失败模式
